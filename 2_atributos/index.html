Atributos de elementos HTML

Os elementos HTML podem ter atributos, que são propriedades visuais, funcionais ou informativas adicionadas aos respectivos elementos. Alguns atributos são específicos de determinados elementos HTML, outros são globais, podem ser usados em qualquer elemento, como o 'style', que define estilo (a cor, formato, dimensões, comportamento de interação, entre outros). O elemento 'body' está com atributo de estilo formatado com largura de 800 pixels, e tudo que está contido dentro do body segue essa formatação.

Imagem

O elemento 'img' possibilita adicionar imagens na página, tanto de arquivos locais como também de imagens online. Para indicar o local do arquivo é usado o atributo 'src'.

Link

O link para outros sites ou lugares na página é feito com o elemento 'a', e a URL do link é especificada com o atributo 'href'. O texto clicável fica entre as tags 'a' e a tag de fechamento '/a'.
